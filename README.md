# Banking System in Java

## 📌 Description
This is a simple console-based banking system developed using Java. It allows users to perform basic banking operations.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)

## 💡 Concepts Used
- Classes and Objects
- Methods
- Conditional Statements
- Loops
- Switch Case

## ▶️ How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
